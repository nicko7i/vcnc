Building vcnc-webÂ¶

All that is needed at this point is:

cd vcnc-web
npm install

vcnc-web is software that runs inside the browser. The build process, using Node.js as a scripting environment, generates a directory which can be served by any web browser as if it were a static site.

% npm run build:production

The build:production target produces optimized and minified JavaScript wrapped in HTML, placing the results in the /dist directory.

During development, the start target builds the project, starts a Node.js server, and opens a connected browser window.

% npm start

We will return to this when we talk about running vcnc-web.
