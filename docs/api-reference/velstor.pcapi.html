velstor.pcapi.mount module¶

class velstor.pcapi.mount.Mount(session, mount_point, **kwargs)[source]¶

Bases: object

The unholy conjunction of a Volume and a Workspace.

Reduces boilerplate when mounting VPs on short-lived workspaces.

Parameters:
  • session – Security information.
  • mount_point – The local directory where the VP will mount.
  • **kwargs – Optional arguments passed to Workspace.

The constructor creates the specified workspace, puts it on the vtrq, and mounts the VP.

Calling dispose un-mounts the vp and removes the workspace from the vtrq.
